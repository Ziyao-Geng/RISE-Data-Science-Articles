# Article 2: Cleaning Clinical Research Data in R

This folder contains the second RISE beginner article.

## Main purpose

Builds on Article 1 and teaches a reproducible data cleaning workflow.

## Files

- `article_02_cleaning_clinical_research_data.qmd`: Quarto article.
- `data_raw/rise_sample_health_data_messy.csv`: Messy fictional teaching dataset.
- `data_cleaned/rise_sample_health_data_cleaned.csv`: Cleaned dataset.
- `scripts/article_02_cleaning_code.R`: Standalone cleaning script.
- `outputs/`: Data cleaning reports.

## How to use

Open the `.Rproj` file, then open the `.qmd` file and click **Render** in RStudio.
