# Article 1: Getting Started with R for Medical Research

This folder contains the first RISE beginner article.

## Main purpose

A true first-time beginner article for students who have never used R before.

## Files

- `article_01_getting_started_with_r.qmd`: Quarto article.
- `data/rise_sample_health_data.csv`: Clean fictional teaching dataset.
- `scripts/article_01_starter_code.R`: Standalone R script containing the main code.
- `outputs/`: Folder for outputs if students save additional work.

## How to use

Open the `.Rproj` file, then open the `.qmd` file and click **Render** in RStudio.
