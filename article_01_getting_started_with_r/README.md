# Article 1: Getting Started with R for Medical Research

This is the first article in the RISE beginner R series. Start here if you are a medical student or clinical researcher who has never used R before.

## What you will learn

- What R, RStudio, and Quarto are.
- How to run your first R commands.
- How to import a CSV file.
- How to inspect rows, columns, and variables.
- How to create simple summaries.
- How to make your first plots with `ggplot2`.

## Open first

| File | Use this when you want to |
| --- | --- |
| [`article_01_getting_started_with_r.html`](article_01_getting_started_with_r.html) | Read the finished article in a browser. |

## Folder contents

| Path | What it contains |
| --- | --- |

| `outputs/` | Empty workspace for any files students create while practising. |

## How to run this article

Open the `.Rproj` file, then open the `.qmd` file and click **Render** in RStudio.

If this is your first time using the repository, read the root [`README.md`](../README.md) as well.
