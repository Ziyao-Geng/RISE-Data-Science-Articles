# RISE Article 01: Getting Started with R for Medical Research

This folder contains a cleaned beginner-friendly Quarto article, a fictional sample dataset, and a starter R script.

## Files

- `article_01_getting_started_with_r_cleaned.qmd`: main Quarto article
- `data/rise_sample_health_data.csv`: fictional teaching dataset
- `scripts/article_01_starter_code.R`: standalone R script containing the main code

## How to use

1. Open RStudio.
2. Create/open an RStudio Project using this folder.
3. Open `article_01_getting_started_with_r_cleaned.qmd`.
4. Run the code chunks step by step, or click Render to produce the HTML article.
