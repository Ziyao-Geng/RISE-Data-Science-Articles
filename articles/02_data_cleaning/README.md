# RISE Article 02: Cleaning Clinical Research Data in R

This folder contains the complete materials for the second beginner RISE data science article.

## Article

- `article_02_cleaning_clinical_research_data.qmd`

This is the Quarto article. Open it in RStudio and render it to HTML.

## Data

- `data_raw/rise_sample_health_data_messy.csv`
- `data_cleaned/rise_sample_health_data_cleaned.csv`

The raw dataset is deliberately messy and fictional. It includes duplicate IDs, inconsistent categories, missing values, and biologically implausible values.

The cleaned dataset is included for convenience, but it can be regenerated by running the Quarto article or the R script.

## Script

- `scripts/article_02_cleaning_code.R`

This is the standalone R script version of the article code.

## Outputs

- `outputs/data_cleaning_summary.csv`
- `outputs/duplicate_participant_report.csv`
- `outputs/missing_value_summary.csv`
- `outputs/suspicious_values_report.csv`
- `outputs/variable_dictionary.csv`

These files document the cleaning process and can be regenerated from the R code.

## Notes for RISE editors

The article is intentionally beginner-friendly and slightly slower than a normal technical tutorial. The goal is not to show the shortest possible code. The goal is to teach students how to think through a reproducible data cleaning workflow.
