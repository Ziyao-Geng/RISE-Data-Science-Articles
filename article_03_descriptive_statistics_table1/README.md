# Article 3: Descriptive Statistics and Table 1 in R

This folder contains the third RISE beginner article.

## Main purpose

Builds on Article 2 and teaches first descriptive analysis using the cleaned dataset.

## Files

- `article_03_descriptive_statistics_and_table1.qmd`: Quarto article.
- `data/rise_sample_health_data_cleaned.csv`: Cleaned fictional teaching dataset.
- `scripts/article_03_descriptive_statistics_code.R`: Standalone analysis script.
- `outputs/`: Summary tables and simple statistical results.
- `figures/`: Figures generated by the article/script.

## How to use

Open the `.Rproj` file, then open the `.qmd` file and click **Render** in RStudio.
