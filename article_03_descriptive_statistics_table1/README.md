# Article 3: Descriptive Statistics and Table 1 in R

This is the third article in the RISE beginner R series. Use it after Article 2 to learn how to describe a cleaned clinical research dataset before formal analysis.

## What you will learn

- How to inspect missing data before analysis.
- How to summarise continuous variables.
- How to summarise categorical variables.
- How to build a simple Table 1.
- How to make exploratory plots.
- How to run simple group comparisons carefully.
- How to export tables, figures, and analysis summaries.

| Path | What it contains |
