# RISE R Beginner Article Series: Articles 1-3

This package contains three beginner-friendly Quarto articles for the RISE website.

## Learning sequence

1. **Article 1 — Getting Started with R for Medical Research**  
   A true first-time beginner article. Students learn what R/RStudio/Quarto are, how to run code, how to import a clean CSV, inspect a dataset, use simple pipes, and make first plots.

2. **Article 2 — Cleaning Clinical Research Data in R**  
   Builds directly on Article 1. Students use a messy version of the sample dataset and learn a reproducible cleaning workflow: raw data preservation, missing values, column names, duplicates, numeric conversion, categorical recoding, suspicious values, and saving cleaned outputs.

3. **Article 3 — Descriptive Statistics and Table 1 in R**  
   Builds on the cleaned dataset from Article 2. Students learn how to summarise continuous and categorical variables, build a simple Table 1, compare groups carefully, and create publication-style exploratory plots.

## Recommended use

Open each article folder as its own RStudio Project, then open the `.qmd` file and click **Render**.

The articles are intentionally separated so that each one can be edited, uploaded, or taught independently.
