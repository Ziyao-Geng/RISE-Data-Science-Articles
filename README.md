# RISE Data Science Articles

This repository contains the source files, datasets, scripts, and planning notes for beginner-friendly RISE data science articles.

The project follows the direction from the 12 May 2026 RISE meeting: start with practical website articles, use a fictional teaching dataset, build toward videos after the first few articles, and later support workshops with reusable teaching resources.

## Folder Guide

- `00_project_planning/`: meeting notes, article roadmap, and future planning material.
- `articles/`: active article projects. Each article has its own Quarto file, data, scripts, outputs, and README where needed.
- `archive/`: older drafts or duplicated project files kept for reference.

## Active Articles

1. `articles/01_getting_started_with_r/`
   - Beginner introduction to R, RStudio, Quarto, loading data, basic summaries, and first plots.

2. `articles/02_data_cleaning/`
   - Practical guide to cleaning a messy clinical research dataset in R.

## Current Roadmap

See `00_project_planning/article_roadmap.md` for the article sequence and next steps.

## Suggested Workflow

1. Draft or edit each article inside its folder under `articles/`.
2. Keep raw or teaching datasets with the article that uses them.
3. Put generated reports, figures, and cleaned datasets in article-specific output folders.
4. Move superseded drafts into `archive/` rather than deleting them immediately.
