# RISE Data Science Articles

This repository contains the source files, datasets, scripts, and planning notes for beginner-friendly RISE data science articles.

## Folder Guide

- `articles/`: active article projects. Each article has its own Quarto file, data, scripts, outputs, and README where needed.

## Active Articles

1. `articles/01_getting_started_with_r/`
   - Beginner introduction to R, RStudio, Quarto, loading data, basic summaries, and first plots.

2. `articles/02_data_cleaning/`
   - Practical guide to cleaning a messy clinical research dataset in R.
